Style guide

The visual contract for every ATLAS surface. Stripe-dense, Koyfin-rich, institutionally restrained. Every page in the product is a composition of the primitives below.

The reading — every visual's fourth wall

Non-negotiable, system-wide. No chart, table, or metric ships without it. This is what makes ATLAS feel like an analyst, not a dashboard.

Every visual carries four zones: (1) the visual itself, (2) a verdict in serif — one sentence saying what the visual means — always visible, (3) explain tabs (How to read · Formula · Source) that expand inline on click, and (4) an action strip — what to do about it — always visible. Zones 2 and 4 are never hidden behind icons. The user should never have to ask "what am I looking at?" or "what do I do with this?"

Both lines are rebased to 100 on 1 Apr 2025 — so every point shows the value of ₹100 invested on that date, not the absolute NAV. The solid green line is the fund; the dashed grey line is the benchmark (Nifty Midcap 150 Total Return Index, which includes reinvested dividends).

When the fund line pulls above the benchmark, the manager is adding value — that gap is alpha. When it converges, the fund is tracking the index. A persistent gap that grows over time (like this one) is the hallmark of repeatable skill rather than luck.

The steepest divergence is Oct '25 → Feb '26, which coincides with the industrials rally — to distinguish skill from luck, switch to the Formula tab to see the Brinson attribution decomposing return into allocation, selection, and interaction effects.

A
For every unit of volatility the fund takes on, it delivers 1.34 units of excess return — materially higher than the 1.12 category average. This fund is being paid well for the risk it carries.

The Sharpe ratio measures return per unit of volatility, net of the risk-free rate.

Sharpe = ( R_fund − R_rf ) / σ_fund With the actual numbers plugged in: = ( 18.4% − 6.8% ) / 8.66% = 11.6% / 8.66% = 1.34

R_fund is the annualised 3Y return (CAGR, actual day count). R_rf is the 3Y average yield on 91-day T-bills (6.8%). σ_fund is the annualised standard deviation of monthly returns over the same 3Y window. All three inputs are available on the "Source" tab with direct links to the JIP table rows.

Design rule: vs benchmark — three canonical patterns
Every quantitative visual must show performance relative to benchmark. This card shows how.
Pattern A — Dual line
Line + area charts, performance charts, NAV series. Fund solid, benchmark dashed grey.
Fund Benchmark
Pattern B — Active weight bar
Bar charts, sector allocation. Zero line = benchmark. Bar color = Brinson contribution sign.
Overweight + right call Wrong call
Pattern C — Reference marker
Scatter plots, bubble charts. Benchmark is a labelled anchor. All dots colored RAG relative to it.
Benchmark anchor Better than bm
Alpha display convention
+9.3pp α — Outperforming (green) +0.7pp α — Within noise band (<1pp) → amber −2.9pp α — Underperforming (red) Always TRI. Always same time window as fund. Never price-only index.
Visualization library stack
Which library ships which kind of chart — lean, not kitchen-sink
Library Ships with Used for Why not others
Recharts React-native charts, declarative Default for 70% of charts: line, area, bar, stacked bar, waterfall, donut, radial Boring by design. Good. We restyle via tokens so it doesn't look like every other Recharts dashboard.
visx Airbnb's D3 + React primitives Factor radar, custom bubble charts, violin plots, small multiples grids When Recharts is too rigid but D3-from-scratch is too much.
D3.js Raw SVG + data binding Treemap, correlation heatmap, force-directed network, horizon charts, Sankey Only where visx doesn't have a primitive. Custom = last resort.
react-three-fiber Three.js bindings for React 3D correlation surfaces, yield curve surfaces, multi-dimensional factor explorers Reserved for the one hero 3D view per product area. 3D is a taste bomb — use sparingly.
AG Grid Enterprise data grid Tables with >1k rows, multi-column sort, pivot, grouping, CSV export Our .table primitive handles anything under 500 rows; above that, AG Grid.
Apache ECharts Powerful all-rounder Not using — Koyfin uses it, but it's hard to theme to our token system without fighting it. Mentioned because you'll see it elsewhere. We're explicitly not shipping this.

Charts & the signature insight component

Charts are restrained: no borders, baseline gridline only, body-type axis labels. The "Atlas noticed" component is the sentient lever — editorial, quiet, persistent.
