MF Rank

Cross-sectional ranking of mutual funds within SEBI categories using four independently scored factors: Returns, Risk, Resilience, Consistency. Each factor is z-scored within category, converted to a 0–100 percentile score via Φ (normal CDF), then averaged into a composite. Tie-break order: Consistency → Risk → Returns → Resilience.
Returns Excess return vs benchmark: 1Y, 3Y, 5Y weighted. Alpha persistence via rolling 12-month alpha. Weight: 25%
Risk Volatility (3Y), downside deviation, max drawdown within category. Lower is better — score is inverted. Weight: 25%
Resilience Downside capture ratio, worst rolling 6-month return. How well the fund holds up in adverse markets. Weight: 25%
Consistency % of rolling 12-month periods beating the benchmark. Measures repeatability of outperformance over time. Weight: 25%
Tie-break order → Consistency → Risk → Returns → Resilience When composite scores are equal, rank by Consistency first

Methodology — 4-Factor Composite Ranking Formula

Show / hide
Factor scoring. Each raw factor metric (excess return, volatility, drawdown, etc.) is cross-sectionally z-scored within the fund's SEBI category, then converted to a 0–100 percentile via the standard-normal CDF (Φ). This ensures comparisons are always within-category, not cross-category.

Returns factor aggregates excess returns at 1Y, 3Y, and 5Y horizons plus a rolling 12-month alpha signal. Higher excess return → higher score.

Risk factor aggregates 3-year annualised volatility, downside deviation, and 3-year maximum drawdown. Lower risk → higher score (metric is inverted before z-scoring).

Resilience factor uses downside capture ratio and worst rolling 6-month return. A fund that holds its ground in down markets scores higher.

Consistency factor measures the percentage of rolling 12-month windows in which the fund beat its category benchmark. It is the tie-break dimension because sustained outperformance is harder to fake than a single strong year.

Composite is the simple unweighted average of the four percentile scores. All four factors receive equal 25% weight. The composite is reported to one decimal place.

Tie-break order: When two funds share an identical composite score, they are ranked by Consistency first, then Risk, then Returns, then Resilience. Within any still-tied pair, alphabetical scheme name is the final decider. 
z_cat(x) = (x − μ_category) / σ_category

score_i(x) = round(100 × Φ(z_cat(x)))      where Φ is the standard-normal CDF

Returns_score     = score(excess_ret_1y, excess_ret_3y, excess_ret_5y, rolling_12m_alpha)
Risk_score        = score(−vol_3y, −downside_dev_3y, −max_dd_3y)        ← inverted
Resilience_score  = score(−downside_capture, −worst_rolling_6m)          ← inverted
Consistency_score = score(rolling_pct_beating_bench)

Composite = (Returns + Risk + Resilience + Consistency) / 4

Tie-break: Consistency → Risk → Returns → Resilience → scheme_name (alpha)
