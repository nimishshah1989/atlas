## Domain constraints

- Read `frontend/mockups/breadth.html` before writing any component.
- Read `frontend/mockups/assets/breadth.js` before writing selector components.
- Read `node_modules/next/dist/docs/` for Next.js 16 API before writing any page.
- Universe and indicator state lives in `page.tsx` — passed down to all block components as props.
  DO NOT use global window state or context for this — React props suffice.
- SWR key format: `["/api/v1/stocks/breadth", universe, "counts"]` (array key) so each universe value
  produces a unique cache entry.
- `data-v2-derived="true"` blocks (oscillator sub-panel, conviction halo, signal playback) may render
  derived content or `<EmptyState title="Coming soon" />` — do NOT wire them to a backend endpoint.
- OscillatorPanel must use Recharts `ComposedChart` with dual Y-axes.
- All chart zone bands (overbought/oversold) must use Recharts `ReferenceArea` with appropriate fill colours.
