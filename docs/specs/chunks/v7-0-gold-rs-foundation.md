## GoldRSService contract

```python
class GoldRSService:
    async def get_gold_price_series(
        self, session, start: date, end: date,
        currency: Literal["INR", "USD"],
    ) -> list[tuple[date, Decimal]]: ...

    def compute_rs_vs_gold(
        self,
        instrument_series: list[tuple[date, Decimal]],
        gold_series: list[tuple[date, Decimal]],
        periods_days: dict[str, int],  # {"1m":21,"3m":63,"6m":126,"12m":252}
    ) -> dict[str, Decimal | None]: ...

    def compute_gold_rs_signal(
        self, rs_benchmark: Decimal | None,
        rs_gold: Decimal | None,
        gold_missing: bool,
        yesterday_age_days: int,
    ) -> str: ...  # AMPLIFIES_BULL|AMPLIFIES_BEAR|NEUTRAL_BENCH_ONLY|FRAGILE|STALE
```

Decimal only. Never `float`. Null when either series lacks data (never 0, never NaN).

## Signal derivation (from contracts/gold-rs-block.md)

```
if gold_missing and yesterday_age_days > 2:      STALE
elif bench > 0 and gold > 0:                     AMPLIFIES_BULL
elif bench < 0 and gold < 0:                     AMPLIFIES_BEAR
elif bench > 0 and gold < 0:                     NEUTRAL_BENCH_ONLY
else:                                            FRAGILE
```

Zero-boundary: treat `0` as non-positive for bull and non-negative for bear (use strict `>` / `<`, fall through to FRAGILE on exact zeros).
