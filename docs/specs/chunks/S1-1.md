## Objective

Convert `frontend/mockups/today.html` into a live Next.js page at `/pulse`. Every
`data-endpoint` block in the mockup becomes a React component that fetches from the
backend via the `useAtlasData` hook created in S1-0. No new backend routes — all
endpoints already exist after V2FE-1/1a/1b/1c.
