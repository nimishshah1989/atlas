## Domain constraints

- Read `frontend/mockups/mf-detail.html` before writing any component.
- Read `frontend/mockups/assets/mf-detail.js` before writing page logic.
- Read `node_modules/next/dist/docs/` for Next.js 16 dynamic route API.
- `params.id` from Next.js App Router is the mstar_id slug (e.g. "ppfas-flexi-cap-direct-growth").
- Category-gated peers: `useAtlasData(category ? "/api/v1/query" : null, peerParams)` — same pattern as S1-4.
- V1FE-9 kill-list: do NOT include "Atlas Verdict", "HOLD / ADD ON DIPS", "STRONG BUY" badges.
  The product decision was to remove these subjective signals.
- All rolling return values: `formatPercent` with sign. Positive green, negative red.
- All monetary values (NAV, AUM): `formatCurrency` — never float + "₹" inline.
- NavChartBlock events overlay: use Recharts `ReferenceLine` with vertical orientation + label.
