**Slice:** `docs/specs/slices/s2-drill-spine.md` (read §3, §4, §5)
**Status:** PENDING  •  **Model:** sonnet  •  **Estimate:** $4 / 4h
**Depends on:** S2-0

## Mission

Build the 5 shared React primitives every S2 view reuses. No page routes yet — just primitives + unit tests + Storybook-style fixture renders. Frontend consumes S2-0's endpoints via the existing `useAtlasData` hook (ref S1-0).

## Scope

1. **`<FourLensRRG rows={...} xLens="rs" yLens="momentum" sizeLens="volume" colorLens="breadth" />`** — 4-lens RRG per slice §3.4. Bubble-size from volume, bubble-color from breadth (green ≥60, amber 40–60, red <40, per §8 Q4). Click handler returns `{scope, id}`. Uses Recharts ScatterChart or D3 — pick one, document the choice.

2. **`<InstrumentTable rows={...} mode="stock|etf|mf" onRowClick={...} onSizeIt={...} />`** — 11-column schema locked in slice §5. Order must match §5 exactly. Sortable per column. "Size it" button per row emits instrument payload.

3. **`<RegimeGauge value={0..100} bands={...} />`** — composite 0-100 gauge used in Cockpit India band. Plus 5 stacked mini-gauges (slice §4.2).

4. **`<SignalPill type="ENTRY|EXIT|REGIME|WARN|CONFIRM" lens?="rs|mom|breadth|volume" />`** — color + icon per signal type. Used in every table row + every 4-lens card.

5. **`<ConvictionDrawer instrument={...} open onClose />`** — skeleton only (slide-in 400px from right, sticky, non-blocking). S2-6 fills in the real content + POST wiring. This chunk just ships the shell + open/close + accessibility.
